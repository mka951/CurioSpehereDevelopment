Introducing,
CurioSphere.

CurioSphere is a website originally created by Matteo Kenzie Rana. The main goal of this website is to create a place to share information and stories about cultures around the world. Inherited culture is not only about dance, language, and so on. But culture is also related to meaning, ways of thinking, and philosophy in it which relate to human harmony with nature.

How CurioSphere can change our world?

Matteo believes that if we work together, if we can create community, then we can protect, and share our culture throughout the world. We can also learn every meaning in it and deep philosophy related to humans and nature. The community that is built can create big changes for the world. That's because we want to understand each other, and create harmony.
